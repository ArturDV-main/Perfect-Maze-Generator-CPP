Maze v1.0

О программе

Maze v1.0 позволяет генерировать идеальный лабиринт по алгоритму эйлера и проходить его от одной точко до другой.

Программа позволяет:

    • Генерировать идеальный лабиринт
    • Настраивать размер лабиринта
    • Сохранять в файл лабиринт
    • Загружать из файла лабиринт
    • Устанавливать начало и конец траектории
    • Отображать траекторию

Установка программы

Для того, чтобы начать использовать Maze вам нужно использовать команду make install.

Открыть программу

Для того, чтобы открыть Maze вам нужно использовать команду make open.

Удаление программы

Для удаления программы вы можете использовать команду make uninstall.
